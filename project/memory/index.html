Evasive Memories

A web application that parallels our need to store data via technology and our ability to remember. This application was then accessible to the public during the 2017 open days of the ESADHaR.

Users have the option of recording short messages in text format accessible to everyone. Each message has a defined lifespan that can be influenced by interaction with users. The degree of corruption is defined by how often the message is read. The more a message is read, the longer it will stay readable. The memories unfold in an infinite space and rearrange themselves in the form of constellations or synapses.
